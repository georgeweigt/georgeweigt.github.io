\section*{Hydrogen atom 2}

In the previous section we used
\begin{equation*}
a_0=\frac{4\pi\epsilon_0\hbar^2}{\mu e^2}
\end{equation*}

Alternatively, $a_0$ can be left undefined by using the Hamiltonian
\begin{equation*}
H=-\frac{\hbar^2}{2\mu}\nabla^2-\frac{\hbar^2}{\mu a_0r}
\end{equation*}

For the new $H$ the eigenvalues are
\begin{equation*}
E_n=-\frac{1}{2\mu}\left(\frac{\hbar}{a_0}\right)^2\frac{1}{n^2}
\end{equation*}
